# matrix-1

## Determinant:

#### Determinant of a 2x2 Matrix

For a matrix:

```css
| a  b |
| c  d |
```

The determinant is calculated as:

```css
det(A) = a*d - b*c
```

#### Determinant of a 3x3 Matrix

For a matrix:

```css
| a  b  c |
| d  e  f |
| g  h  i |
```

The determinant is calculated using the formula:

```css
det(A) = a*(e*i - f*h) - b*(d*i - f*g) + c*(d*h - e*g)
```
